发表于 2016-05-09   |   分类于 Android   |     |   本文阅读量 次

GreenDao 是在 Android 开发中广泛使用的数据库框架，其优点是运行效率高，内存消耗少，性能佳（没有考证，网上说的😅）
其它一些常用的数据库框架有：

  • LitePal
  • ActiveAndroid
  • OrmLite
  • realm

目前还在更新中的有 LitePal，realm。本篇博客只讲 GreenDao 相关知识
